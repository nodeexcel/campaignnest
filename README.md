# CampaignNest ![Build Status](https://img.shields.io/badge/build-passing-brightgreen) ![Version](https://img.shields.io/badge/version-1.0.0-blue)

## Project Description
CampaignNest is a beginner-friendly web application designed to help small businesses and startups automate their marketing campaigns. Users can easily create, customize, and launch email campaigns while tracking their performance through a straightforward analytics dashboard.

## Features
- 🛠️ User-friendly campaign creation wizard
- 🎨 Basic email template customization
- 📊 Simple analytics dashboard for campaign performance

## Tech Stack
### Backend
- Node.js 🟢

### Database
- MongoDB 🐬

## Installation
To set up CampaignNest locally, follow these steps:

- Clone the repository:
  bash
  git clone https://github.com/nodeexcel/campaignnest.git
  - Navigate into the project directory:
  bash
  cd campaignnest
  - Install the required dependencies:
  bash
  npm install
  - Set up your MongoDB database and update the connection string in the configuration file.

- Start the application:
  bash
  npm start
  ## Usage
Once the application is running, navigate to `http://localhost:3000` in your web browser. You can start creating your marketing campaigns using the intuitive interface.
